# 数据后台系统

> 对超市数据进行总体分析、分时段分析、提取结算等

## 如何使用

``` bash
# 安装
$ npm install

# 在开发环境运行
$ npm run dev

# 在生产环境运行
$ npm run build
$ npm run start
```
